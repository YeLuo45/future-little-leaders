# 任务标签体系说明文档

## 概述

本文档详细说明亲子任务应用中的任务标签体系，包括各分类的教育目标、具体标签及实践建议。任务标签体系旨在为父母提供科学、全面的儿童发展引导框架，通过多元化的任务设计促进孩子全面成长。

## 标签体系结构

目前，应用包含六大分类标签，每个分类聚焦儿童发展的不同关键领域：

1. 生活习惯类：培养基础生活能力和时间管理
2. 学习思维类：发展认知能力和解决问题的思维方式  
3. 心理品德类：建立健康情绪和道德价值观
4. 社会适应类：提升社会适应力和时代素养
5. 社交与运动类：增强人际交往能力和身体素质
6. 财商培养类：建立正确金钱观和财富管理意识

## 财商培养（金融素养）详解

### 核心教育目标

财商培养分类的核心目标是建立孩子正确的金钱观、消费观、财富创造与责任意识，涵盖金钱认知、规划、实践三大维度。通过系统性的财商教育，帮助孩子从小建立健康的财富观念。

### 二级标签说明

财商培养分类包含以下二级标签，分别对应财商教育的不同方面：

#### 金钱认知
- **金钱认知**：认识纸币/硬币、了解不同国家货币，建立基础金钱概念
- **价值理解**：理解劳动换取报酬、商品价格与价值的关系
- **需求区分**：学会辨别"需要"vs"想要"，如"买文具"vs"买玩具"

#### 财富管理
- **记账习惯**：记录零花钱收支，创建"每日消费小账本"
- **储蓄规划**：设置"梦想基金"、"应急基金"等分类储蓄
- **理性消费**：制定购物清单、学习比较价格再购买

#### 财富创造
- **劳动换酬**：通过家庭任务赚取"虚拟货币"，建立劳动创造价值的认知
- **创意变现**：尝试旧物义卖、手工作品交换等活动
- **投资启蒙**：通过种植植物观察"成长收益"、简单基金概念游戏等

#### 社会责任
- **公益捐赠**：引导孩子用零花钱帮助他人，如"给山区孩子买书"
- **可持续消费**：培养选择环保商品、拒绝过度包装的消费习惯

### 任务设计建议

1. **阶段性安排**：根据孩子年龄设计合适的财商任务
   - 3-5岁：认识钱币、辨别价格
   - 6-9岁：记账练习、储蓄习惯培养
   - 10岁以上：简单投资概念、社会责任意识

2. **生活化实践**：将财商教育融入日常生活
   - 逛超市时让孩子比较商品价格
   - 过节收到压岁钱时制定储蓄计划
   - 设立家庭小任务清单，完成后获得相应报酬

3. **情境化学习**：创设实际金钱使用场景
   - 家庭内部"小市场"，练习买卖交易
   - "家庭银行"，体验存取款和利息概念
   - 慈善捐款活动，体验金钱的社会价值

## 任务标签使用指南

1. **多元组合**：鼓励跨分类选择标签，如一个任务可同时具有"储蓄规划"和"责任感"标签

2. **渐进提升**：建议从简单标签开始，随着孩子成长逐步引入复杂概念标签

3. **个性定制**：根据孩子特点和家庭价值观选择侧重点

4. **定期回顾**：定期检视已完成任务的标签分布，确保孩子成长的均衡性
