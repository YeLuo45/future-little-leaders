# 多端图表方案选型与实践

## 选型建议
- 复杂交互、功能丰富：优先 ECharts + echarts-for-uniapp（官方维护，社区成熟，兼容H5/小程序/APP）
- 性能优先、极致轻量：可选 VChart（@visactor/vchart-uni）
- 纯小程序简单场景：UCharts

## 本项目实践
- 采用 echarts-for-uniapp，统一多端图表体验。
- 通过 option 响应式驱动，数据变化自动刷新。
- 组件注册与用法详见 docs/api/echarts-for-uniapp.md

## 性能与兼容性建议
- 小程序端避免大数据量、复杂渐变，必要时做数据抽样。
- 图表组件按需懒加载，避免页面初次渲染阻塞。
- 统一样式变量，保证多端视觉一致。
