# 微信小程序育儿应用

## 项目简介

这是一个基于uni-app框架开发的微信小程序，专注于为年轻父母提供育儿经验分享、任务管理和积分商城功能。应用采用Vue 3和组合式API开发，具有良好的性能和用户体验。

## 主要功能

- **用户认证**: 登录、注册、找回密码等基础账户功能
- **任务管理**: 创建和管理每日育儿任务，完成后获得积分奖励
- **积分商城**: 使用积分兑换育儿物品和服务
- **社区交流**: 育儿经验分享、问答和交流平台
- **个人中心**: 用户资料管理、积分记录、收藏内容等

## 技术栈

- **前端框架**: uni-app, Vue 3, Vuex
- **UI组件**: uni-ui
- **网络请求**: uni.request, axios
- **存储**: uni-storage
- **图表**: uCharts

## 社区功能

育儿社区模块为用户提供一个分享育儿经验和知识的平台，主要特点：

- **分类浏览**: 支持按育儿经验、教育方法、营养健康等多种分类查看内容
- **搜索功能**: 快速查找感兴趣的主题和内容
- **互动功能**: 支持点赞、评论、回复等互动方式
- **内容创建**: 用户可发布主题、分享经验和提问
- **通知提醒**: 实时获取回复和点赞通知
- **暗黑模式**: 支持深色界面，保护用户夜间使用时的视力健康

## 页面展示

### 登录页面
登录页采用紫色背景设计，包含用户名和密码输入框，以及记住我选项、登录按钮、忘记密码和注册账号链接。支持暗色模式显示。

### 注册页面
注册页面与登录页保持一致的设计风格，包含用户名、密码和确认密码输入框，以及同意条款选项和注册按钮。同样支持暗色模式显示。

### 首页（任务大厅）
首页展示用户任务列表，每个任务以卡片形式呈现，含任务名称、标签、创建时间和完成进度条。任务支持彩色标签分类（如专注力、财商、学习能力等），按未完成优先并按时间降序排序。用户可以通过右上角的删除按钮删除任务，也可以点击完成打卡按钮标记任务完成并获得积分。页面底部设有导航栏可切换不同页面，右下角的悬浮添加按钮用于创建新任务。在暗色模式下，界面自动调整为深色背景和适当的文本颜色。

### 积分商城页面
积分商城页面顶部显示用户当前积分，包含搜索框用于过滤商品。主体区域展示精心设计的商品卡片，采用白色背景搭配阴影效果，提升立体感。每张卡片左侧为商品图标，右上角小标签显示库存数量。卡片内容区域包含商品名称、积分标签和兑换按钮，点击时具有触感反馈效果。点击右下角悬浮添加按钮可打开添加商品表单模态框，支持选择商品图标、设置名称、积分和库存，所有商品数据会自动保存到本地存储。底部导航栏可切换至其他页面。在暗色模式下，商品卡片和界面元素采用深色背景，并调整文本颜色以保证可读性。

### 个人中心页面
个人中心页面顶部展示用户头像和基本信息卡片，下方是用户数据面板，显示任务完成数、积分余额和商品兑换次数等统计信息。中间区域展示任务和兑换操作记录，用卡片形式呈现近期活动。底部是功能菜单，包括设置、客户服务等选项。页面同样支持暗色模式，在夜间使用时可自动切换至适合的暗色界面，保护用户视力。

## 暗色模式实现

本项目实现了完整的暗色模式支持，具有以下特点：

1. **全局主题管理**：通过`themeUtils.js`管理全局主题状态，提供获取、切换和设置主题的API
2. **本地存储**：用户的主题偏好会保存在本地存储中，下次启动时自动应用
3. **动态样式切换**：使用条件类名（`:class="{'dark-mode': isDarkMode}"`）实现动态主题切换
4. **一致的设计语言**：暗色模式下保持统一的色彩系统，包括背景色、文本颜色、边框和阴影
5. **适应环境**：主色调在暗色模式下调整为更柔和的色彩，避免过于刺眼
6. **全应用支持**：从登录页到应用内的所有功能页面均支持暗色模式

## 快速开始

### 环境准备

- Node.js (>= 12.x)
- 微信开发者工具
- HBuilderX（推荐）

### 安装依赖

```bash
npm install
```

### 开发模式

使用HBuilderX打开项目，选择"运行"菜单下的"运行到小程序模拟器"，或者选择"发行"菜单下的"小程序-微信"。

也可以使用命令行：

```bash
# 开发模式 - 微信小程序
npm run dev:mp-weixin

# 构建生产版本 - 微信小程序
npm run build:mp-weixin
```

## 设计说明

- 采用紫色(#8477fa)作为主题色，营造现代简约的视觉风格
- 使用圆形装饰元素增强页面层次感和视觉美感
- 表单设计采用白色卡片式布局，提高可读性
- 输入框添加图标提示，提升用户体验
- 响应式设计，使用rpx单位适配各种屏幕尺寸
- 登录和注册页面保持一致的设计语言，提高品牌识别度
- 暗色模式下使用深色背景（#121212/#1f1f1f）并调整文本颜色以保证可读性

## 开发规范

- 组件名使用PascalCase命名
- CSS类名使用kebab-case命名
- 变量和函数使用camelCase命名
- 代码缩进使用2个空格
- 所有页面必须有注释说明
- 页面间导航统一使用绝对路径 (如: '/pages/login/login')
