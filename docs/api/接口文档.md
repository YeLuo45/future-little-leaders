# 接口文档

## 用户认证接口
### 账号密码登录
- 接口地址：/api/auth/login
- 方法：POST
- 请求参数：
  | 参数名   | 类型   | 必填 | 说明         |
  |----------|--------|------|-------------|
  | username | string | 是   | 用户名       |
  | password | string | 是   | 密码(MD5加密) |
  | remember | boolean| 否   | 记住登录状态 |

- 响应示例：
```json
{
  "error": 0,
  "body": {
    "token": "jwt_token",
    "expire": 3600
  }
}
```
- 错误代码：
  | 代码 | 类型     | 说明           |
  |------|----------|---------------|
  | 400  | 参数错误 | 用户名密码为空 |
  | 401  | 认证失败 | 用户名密码错误 |

### 短信验证码登录
- 接口地址：/api/auth/sms-login
- 方法：POST
- 请求参数：
  | 参数名 | 类型   | 必填 | 说明         |
  |--------|--------|------|-------------|
  | phone  | string | 是   | 手机号       |
  | code   | string | 是   | 6位验证码    |

- 响应示例：同密码登录
- 错误代码新增：
  | 代码 | 类型     | 说明           |
  |------|----------|---------------|
  | 406  | 验证失败 | 验证码错误/过期 |
  | 415  | 加密错误 | 解密失败/无效密文 |
  | 416  | 密钥错误 | 无效密钥版本 |

### 微信一键登录
- 接口地址：/api/auth/wechat-login
- 方法：POST
- 请求参数：
  | 参数名 | 类型   | 必填 | 说明          |
  |--------|--------|------|--------------|
  | code   | string | 是   | 微信授权code  |

- 响应示例：同密码登录

### 注册账号
- 接口地址：/api/auth/register
- 方法：POST
- 请求参数：
  | 参数名   | 类型   | 必填 | 说明         |
  |----------|--------|------|-------------|
  | username | string | 是   | 用户名       |
  | password | string | 是   | AES加密后的密文(Base64编码) |
  | algorithm | string | 是   | 加密算法类型(例：AES/CBC/PKCS5Padding) |
  | keyVersion | string | 是   | 密钥版本号 |
  | iv | string | 是   | 初始化向量(Base64编码) |
  | phone    | string | 是   | 手机号       |
  | smsCode | string | 是   | 短信验证码   |

- 错误代码新增：
  | 代码 | 类型     | 说明               |
  |------|----------|-------------------|
  | 409  | 资源冲突 | 用户名/手机号已注册 |

### 密码重置

## 积分相关接口
### 积分记录分页查询
- 接口名称：分页获取积分记录
- 接口地址：/api/points/history
- 方法：GET
- 请求参数：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|-----|
| page | integer | 否 | 当前页码，默认1 |
| pageSize | integer | 否 | 每页数量，默认10 |
| type | string | 否 | 记录类型(income/expense) |
| startDate | string | 否 | 起始日期(yyyy-MM-dd) |
| endDate | string | 否 | 结束日期(yyyy-MM-dd) |

- 响应示例：
```json
{
  "error": 0,
  "body": {
    "list": [
      {
        "createdAt": "2024-03-15T14:30:00",
        "type": "任务奖励",
        "points": 10,
        "balance": 160,
        "taskName": "每日阅读"
      }
    ],
    "pagination": {
      "total": 35,
      "page": 1,
      "pageSize": 10
    }
  }
}
```
- 错误代码：
| 代码 | 类型 | 说明 |
|------|------|-----|
| 400 | 参数错误 | 日期格式不正确 |
| 412 | 验证失败 | 日期范围超过3个月 |
**通用请求头**
- Authorization: Bearer <token>
- X-Requested-With: XMLHttpRequest
- X-Anti-CSRF-Token: <csrf_token>

### 获取宝宝积分
- 接口名称：获取宝宝积分
- 接口核心功能描述：获取指定宝宝的当前积分余额
- 接口地址：/api/points/{babyId}
- 方法：GET
- 需要登录：是
- 请求头：
  - Authorization: Bearer <token>
- 请求参数：babyId（string，必填）：宝宝ID（路径参数）
- 响应类型：json
- 响应示例：
```json
{
  "error": 0,
  "body": {"points": 150},
  "message": ""
}
```
- 错误代码：
  - 401：未登录（需跳转登录页）
  - 404：宝宝信息不存在

### 添加宝宝积分
- 接口名称：添加宝宝积分
- 接口核心功能描述：为指定宝宝增加积分（通常因完成任务）
- 接口地址：/api/points/add
- 方法：POST
- 需要登录：是
- 请求头：
  - Authorization: Bearer <token>
- 请求参数（JSON Body）：
  - babyId（string，必填）：宝宝ID
  - points（number，必填）：积分值（必须大于0）
  - description（string，选填）：积分描述
- 响应类型：json
- 响应示例：
```json
{
  "error": 0,
  "body": {"totalPoints": 200},
  "message": ""
}
```
- 错误代码：
  - 400：积分值无效
  - 401：未登录
  - 403：无操作权限

## 任务相关接口

### 创建任务
- 接口地址：/api/tasks
- 方法：POST
- 请求头：
  - Authorization: Bearer <token>
  - X-Requested-With: XMLHttpRequest
  - X-Anti-CSRF-Token: <csrf_token>
- 请求参数（JSON Body）：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|-----|
| title | string | 是 | 任务标题（2-20字） |
| description | string | 否 | 任务描述（≤50字） |
| tags | string[] | 是 | 任务标签（≤3个） |
| type | string | 是 | 任务类型：normal(普通)/recurring(周期) |
| points | number | 是 | 任务积分（≥1） |
| babyId | string | 是 | 关联的宝宝ID |
| cycleParams | object | 否 | 周期任务专属参数 |

**cycleParams参数**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|-----|
| recurringType | string | 是 | daily(每日)/weekly(每周)/custom(自定义) |
| weekdays | number[] | 条件必填 | 1-7对应周一到周日（当recurringType=weekly时必填） |
| customStartTime | string | 条件必填 | 开始时间（HH:mm，当recurringType=custom时必填） |
| customEndTime | string | 条件必填 | 结束时间（HH:mm，当recurringType=custom时必填） |

- 响应示例：
```json
{
  "error": 0,
  "body": {
    "taskId": "TASK_123456",
    "nextResetTime": "2024-03-20T00:00:00"
  }
}
```
- 错误代码：
| 代码 | 类型 | 说明 |
|------|------|-----|
| 400 | 参数错误 | 必填参数缺失/格式错误 |
| 403 | 权限不足 | 无操作指定宝宝的权限 |
| 406 | 业务错误 | 标签超限/周期参数冲突 |
| 412 | 验证失败 | CSRF token验证失败 |
| 415 | 加密错误 | 宝宝ID解密失败 |
