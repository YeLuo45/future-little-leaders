# 未来小领袖 - 亲子育儿助手

未来小领袖是一款专为家长和孩子设计的任务管理和积分奖励应用。通过游戏化的方式，帮助家长培养孩子的自律能力和责任感，同时增强亲子互动。

## 功能特点

- **宝宝管理**: 支持添加多个宝宝，每个宝宝有独立的积分账户和任务记录
- **任务系统**: 家长可以创建日常、周期和一次性任务，设置任务积分
- **积分商城**: 孩子可以用完成任务获得的积分兑换奖励
- **数据统计**: 直观展示孩子的任务完成情况和积分变化
- **主题切换**: 支持明暗两种主题模式

## 技术栈

- 前端框架: Vue 3 + Uni-App
- 状态管理: Pinia
- 样式: SCSS
- 打包工具: Vite
- 多端支持: H5、微信小程序、支付宝小程序等

## 代码质量规范

### 微信小程序规范

为确保小程序通过微信的代码质量检测，我们需要注意以下几点：

1. **代码包大小**
   - 主包大小不超过2MB
   - 分包大小不超过2MB
   - 总计不超过16MB

2. **媒体资源**
   - 单个图片或音频资源不超过200KB
   - 对于较大图片，使用图片压缩工具减小体积或使用CDN
   - 图片资源使用WebP等高压缩比格式

3. **代码注入优化**
   - 使用 `lazyCodeLoading: "requiredComponents"` 开启组件按需注入
   - 避免在全局声明使用率低的自定义组件
   - 及时移除JSON中未使用的自定义组件声明

4. **插件使用**
   - 不建议引用过多大型插件
   - 需要时将插件置于分包，通过分包异步化方式引入

这些优化可以显著提升小程序的启动性能、运行效率和用户体验。

### 图片资源压缩工具

项目提供了自动图片压缩工具，可以自动处理超过200KB的图片资源：

```bash
# 安装依赖（如果尚未安装）
npm install sharp fs-extra --save-dev --legacy-peer-deps

# 运行图片压缩脚本
npm run compress-images
```

压缩脚本特性：
- 自动检测并压缩超过200KB的图片文件
- 压缩前自动备份原始图片到backup目录
- 保持较高的图片质量（70%）同时大幅减小体积
- 重新调整图片尺寸（最大宽度800px）
- 支持PNG、JPG、JPEG、WebP等常见格式
