# 微信小程序 - UniApp Vue3 项目

## 项目概述

这是一个基于UniApp + Vue3开发的微信小程序项目，采用现代化UI设计，提供了用户登录、注册、任务管理和积分商城等功能。项目使用紫色(#8477fa)作为主题色，创建了美观且用户友好的界面，同时支持暗色模式，提升用户体验。

## 快速开始

### 环境准备

- Node.js (>= 12.x)
- 微信开发者工具
- HBuilderX（推荐）

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
# 开发模式 - 微信小程序
npm run dev:mp-weixin

# 构建生产版本 - 微信小程序
npm run build:mp-weixin
```

## 主要功能

- ✅ 登录页面 - 账号密码登录，记住我选项
- ✅ 注册页面 - 账号密码注册，同意条款选项
- ✅ 任务管理 - 创建、编辑、完成和删除任务，标签分类
- ✅ 积分商城 - 积分兑换商品，搜索商品，管理商品
- ✅ 个人中心 - 用户信息展示，操作记录，设置选项
- ✅ 暗色模式 - 全应用支持暗色/亮色主题切换
- 🔄 忘记密码功能（开发中）

## 技术特点

- 使用Vue3组合式API开发，提高代码复用性和可维护性
- 采用UniApp跨平台框架，支持多端部署
- 实现响应式设计，适配不同屏幕尺寸
- 使用本地存储实现数据持久化
- 支持暗色模式，提升用户体验
- 模块化的项目结构，便于团队协作开发
