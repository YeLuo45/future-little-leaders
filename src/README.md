# 育儿时光纪应用

育儿时光纪是一款面向家庭的亲子任务管理应用，旨在通过游戏化的任务系统，帮助孩子培养良好的习惯，发展各项能力，成为未来的小领袖。

## 功能特点

### 宝宝管理
- 支持添加多个宝宝账号
- 每个宝宝独立的任务和积分系统
- 个性化宝宝头像和信息设置

### 任务系统
- 支持一次性任务和周期性任务
- 任务标签分类和搜索功能
- 任务完成后自动积分奖励

### 积分奖励
- 完成任务获得积分
- 积分可兑换家长设定的奖励
- 积分历史记录和统计

### 认证系统
- 支持密码验证，支持明文/密文切换
- 支持指纹识别（仅限支持设备）
- 支持人脸识别（仅限支持设备）
- 确保敏感操作的安全性

## 技术栈

- 前端框架: Vue 3 + uni-app
- UI组件: 自定义组件
- 存储: 本地存储 + 云存储
- 认证: 微信小程序SOTER安全系统

## 开发指南

### 环境要求
- Node.js 14.0+
- HBuilderX 3.0+

### 安装依赖
```bash
npm install
```

### 运行开发服务
```bash
npm run dev:mp-weixin
```

### 构建生产版本
```bash
npm run build:mp-weixin
```
