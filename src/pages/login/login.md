# 变更日志（login.vue）

## 2024-03-xx

### 新增
- 登录页支持三种方式：账号密码登录、微信一键登录、手机验证码登录。
- 增加tab切换，用户可自由选择登录方式。
- 手机验证码登录支持mock验证码发送与倒计时，校验通过后mock登录。
- 微信登录按钮mock实现，实际需对接后端。

### 说明
- 所有登录方式均为前端mock，后续可对接真实后端接口。
- 交互体验与UI风格与原有页面保持一致。

---

# 使用说明

1. 用户可在登录页顶部tab切换选择登录方式。
2. 账号密码登录：输入用户名和密码，点击登录。
3. 手机验证码登录：输入手机号，获取验证码，输入验证码后登录。
4. 微信一键登录：点击按钮直接登录（mock）。
5. 登录成功后自动跳转首页。
