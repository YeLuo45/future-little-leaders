# 宝宝管理页面说明

本页面实现了对宝宝信息的管理功能，包括添加、编辑、删除宝宝，以及设置当前活跃宝宝。

## 主要功能

1. **宝宝列表展示**：展示所有已添加的宝宝，包括头像、昵称、年龄和积分信息
2. **宝宝头像管理**：支持选择默认头像或上传自定义头像
3. **宝宝昵称编辑**：允许修改宝宝昵称
4. **宝宝删除**：删除宝宝信息（同时会删除相关任务和积分记录）
5. **设置当前宝宝**：设置某个宝宝为当前活跃宝宝
6. **添加新宝宝**：跳转到添加宝宝页面

## 数据结构

宝宝信息存储在本地存储中的 `babies` 键中，格式为JSON数组：

```javascript
[
  {
    id: 'baby_timestamp',         // 宝宝唯一ID
    name: '宝宝昵称',             // 宝宝昵称
    birthdate: '2020-01-01',      // 出生日期，格式为YYYY-MM-DD
    avatar: '/path/to/avatar.jpg', // 头像路径
    createdAt: '2023-05-12T10:00:00.000Z' // 创建时间
  },
  // ... 其他宝宝
]
```

当前选中的宝宝ID存储在 `currentBabyId` 键中。

## 事件系统

为了实现宝宝信息的实时同步，页面实现了以下事件：

1. **refreshBabyList**：刷新宝宝列表
2. **babyChanged**：切换当前宝宝

当宝宝信息有变更时，会触发这些事件，其他页面监听这些事件可以实时更新显示的宝宝信息。

## 页面交互流程

1. 进入页面时加载所有宝宝信息
2. 点击宝宝头像可以打开头像选择弹窗
3. 点击宝宝昵称旁的编辑按钮可以修改昵称
4. 点击删除按钮可以删除宝宝
5. 点击设置按钮可以将该宝宝设为当前活跃宝宝
6. 点击页面右下角的浮动按钮可以跳转到添加宝宝页面

## 样式说明

页面采用卡片式布局，每个宝宝信息以卡片形式展示。支持明暗两种主题模式，通过CSS变量和类名切换实现主题切换。头像选择弹窗采用全屏遮罩+中心弹出内容的形式。
