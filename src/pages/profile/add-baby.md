# 添加宝宝页面

## 功能介绍

添加宝宝页面（`add-baby.vue`）允许用户添加新的宝宝信息，填写宝宝的姓名、性别、头像和出生日期，保存成功后自动将该宝宝设为当前选中的宝宝。

## 页面结构

1. **顶部导航栏**：包含返回按钮和页面标题
2. **表单内容**：
   - 宝宝姓名：文本输入框
   - 性别选择：男/女单选
   - 头像上传：支持从相册选择或拍照
   - 出生日期：日期选择器
3. **提交按钮**：表单验证通过后可点击提交

## 技术实现

- 使用 Vue 3 组合式 API (`<script setup>`) 实现
- 表单验证使用 `computed` 属性自动监听必填项
- 头像选择使用 `uni.chooseImage` API
- 宝宝添加成功后:
  1. 保存到本地存储 `babies` 数组
  2. 设置该宝宝为当前选中 `currentBabyId`
  3. 通过 `uni.$emit('babyChanged', id)` 广播宝宝变更事件
  4. 通过 `uni.$emit('refreshBabyList')` 通知 profile 页面刷新宝宝列表

## 数据存储

```js
// 单个宝宝数据结构
{
  id: 'baby_时间戳',
  name: '宝宝姓名',
  gender: '男/女',
  avatar: '头像路径',
  birthday: '出生日期'
}
```
