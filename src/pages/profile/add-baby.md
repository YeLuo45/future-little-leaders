# 添加宝宝页面说明

本页面用于添加新的宝宝信息，包括输入宝宝昵称、选择出生日期和设置头像。

## 主要功能

1. **宝宝昵称输入**：输入宝宝的昵称
2. **出生日期选择**：使用日期选择器选择宝宝的出生日期
3. **头像设置**：选择默认头像或上传自定义头像
4. **信息验证**：确保必填信息已填写
5. **保存信息**：将宝宝信息保存到本地存储

## 数据结构

新添加的宝宝信息将以下面的格式添加到本地存储中的 `babies` 数组：

```javascript
{
  id: 'baby_timestamp',         // 宝宝唯一ID
  name: '宝宝昵称',             // 宝宝昵称
  birthdate: '2020-01-01',      // 出生日期，格式为YYYY-MM-DD
  avatar: '/path/to/avatar.jpg', // 头像路径
  createdAt: '2023-05-12T10:00:00.000Z' // 创建时间
}
```

## 页面交互流程

1. 用户输入宝宝昵称
2. 用户选择宝宝出生日期
3. 用户点击头像预览区域，打开头像选择弹窗
4. 用户可以从默认头像中选择一个，或者点击上传按钮上传自定义头像
5. 用户点击"添加宝宝"按钮，系统验证信息
6. 验证通过后，保存宝宝信息并返回上一页
7. 如果这是第一个添加的宝宝，自动设为当前宝宝

## 事件系统

页面成功添加宝宝后会触发 `refreshBabyList` 事件，通知其他页面刷新宝宝列表。

## 表单验证

1. 宝宝昵称和出生日期为必填项
2. 头像为可选项，未设置时使用默认头像
3. 提交时如有必填项未填写，会显示提示信息

## 样式说明

页面采用表单布局，每个表单项按顺序垂直排列。头像选择部分居中显示，显示当前选择的头像预览。页面底部有一个提交按钮。支持明暗两种主题模式。头像选择弹窗采用全屏遮罩+中心弹出内容的形式。
