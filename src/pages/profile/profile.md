# 宝宝管理功能变更日志

## 2024-03-xx

### 变更内容
- 实现多宝宝管理功能，支持一个账号创建、切换多个宝宝
- 新增宝宝选择器，在"我的"页面支持宝宝切换
- 新增独立的"添加宝宝"页面（`add-baby.vue`），包含姓名、性别、头像、出生日期等信息
- 添加全局宝宝切换广播机制，任务、兑换记录等数据自动按当前宝宝隔离

### 数据结构
- `babies`: 宝宝列表，存储格式：
  ```json
  [
    {
      "id": "baby_时间戳",
      "name": "宝宝姓名",
      "gender": "男/女",
      "avatar": "头像地址",
      "birthday": "出生日期"
    }
  ]
  ```
- `currentBabyId`: 当前选中的宝宝ID

### 通信机制
- 切换宝宝时，通过 `uni.$emit('babyChanged', babyId)` 全局广播
- 各页面监听 `babyChanged` 事件，自动切换数据展示

# 使用说明

1. **宝宝切换**
   - 在"我的"页面，点击宝宝选择器可切换当前操作的宝宝
   - 所有任务、积分、兑换记录等数据会自动按当前宝宝过滤

2. **添加宝宝**
   - 点击"添加宝宝"按钮，跳转到添加页面
   - 填写宝宝姓名（必填）、性别（必填）、头像（可选）和出生日期（必填）
   - 确认添加后，系统自动切换到新添加的宝宝

3. **数据隔离**
   - 不同宝宝的任务、兑换记录等数据互相隔离
   - 创建任务时会自动分配给当前选中的宝宝
   - 兑换商品时同样记录当前宝宝信息
