# 变更日志

## [1.4.0] - 2023-10-17

### 重构

- 移除 `pointsManager.js`，统一使用 `pointsStore.js` 管理积分状态
  - 修改所有引用 pointsManager 的页面，改用 pointsStore
  - 完善 pointsStore，确保所有功能完整迁移
  - 添加积分数据重新计算功能，确保积分同步准确

### 修复

- 修复完成任务添加积分失败的问题
- 优化积分计算逻辑，确保积分变更正确反映在UI上

## [1.3.0] - 2023-10-16

### 优化

- 完善积分记录系统，显示所有积分变动明细
  - 改进"完成任务"积分记录，显示具体任务名称
  - 改进"兑换商品"扣减积分记录，显示具体商品名称
  - 统一积分记录来源，整合pointsManager和pointsStore

### 修复

- 修复首页完成任务和商城兑换商品的积分记录未正确显示在积分记录页面的问题
- 统一积分管理逻辑，确保所有积分变动都会记录详细的来源信息
