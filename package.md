# package.json 配置说明

## 基础信息
- `name`: 项目名称
- `version`: 项目版本号

## 脚本命令
### 开发命令
- `dev:custom`: 自定义平台开发
- `dev:h5`: H5开发
- `dev:h5:ssr`: H5 SSR开发
- `dev:mp-weixin`: 微信小程序开发
- `dev:mp-alipay`: 支付宝小程序开发
- `dev:mp-baidu`: 百度小程序开发
- `dev:mp-toutiao`: 字节跳动小程序开发
- `dev:mp-qq`: QQ小程序开发
- `dev:mp-jd`: 京东小程序开发
- `dev:mp-kuaishou`: 快手小程序开发
- `dev:mp-lark`: 飞书小程序开发
- `dev:mp-xhs`: 小红书小程序开发
- `dev:quickapp-webview`: 快应用开发
- `dev:quickapp-webview-huawei`: 华为快应用开发
- `dev:quickapp-webview-union`: 联盟快应用开发

## 依赖说明
### 生产依赖
- `@dcloudio/uni-app`: uni-app 核心包
- `@dcloudio/uni-components`: uni-app 组件包
- `@dcloudio/uni-h5`: H5平台支持
- `@dcloudio/uni-mp-weixin`: 微信小程序平台支持
- `@dcloudio/uni-mp-alipay`: 支付宝小程序平台支持
- `@dcloudio/uni-mp-baidu`: 百度小程序平台支持
- `@dcloudio/uni-mp-toutiao`: 字节跳动小程序平台支持
- `@dcloudio/uni-mp-qq`: QQ小程序平台支持
- `@dcloudio/uni-mp-jd`: 京东小程序平台支持
- `@dcloudio/uni-mp-kuaishou`: 快手小程序平台支持
- `@dcloudio/uni-mp-lark`: 飞书小程序平台支持
- `@dcloudio/uni-mp-xhs`: 小红书小程序平台支持
- `@dcloudio/uni-quickapp-webview`: 快应用平台支持
- `vue`: Vue.js 框架
- `vue-i18n`: Vue国际化插件

- `@dcloudio/types`: TypeScript类型定义

- `@dcloudio/uni-cli-shared`: CLI共享工具
- `@dcloudio/uni-stacktracey`: 堆栈跟踪工具
- `@dcloudio/vite-plugin-uni`: Vite插件
- `@vue/runtime-core`: Vue运行时核心
- `vite`: 构建工具
